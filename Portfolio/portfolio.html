Website Deployment

II deployed a basic website using AWS, leveraging services like Amazon S3 and CloudFront. The website files (HTML, CSS, and JavaScript) were uploaded to an S3 bucket, which serves as the storage and hosting solution. To ensure fast global delivery, I configured CloudFront as a Content Delivery Network (CDN) to cache and distribute the content. The website is accessible via a custom domain name, which I managed through Route 53. This deployment is scalable and cost-effective, as S3 and CloudFront automatically handle traffic and storage needs.

Helping Hands

I developed a website called "Helping Hands," aimed at assisting people in need by providing a crowdfunding platform specifically for health-related issues. The site allows individuals to create campaigns for medical expenses, treatments, and surgeries, sharing their personal stories to encourage donations. Users can securely donate via integrated payment systems, and they can track the progress of each campaign. The platform also offers options to share campaigns on social media to reach a wider audience. The website is built with a responsive design, ensuring it works seamlessly on both desktop and mobile devices, making it easy for people to contribute and support those in need.
